# 🎥 Advanced Telegram YouTube Downloader Bot

A production-ready Telegram bot for downloading YouTube videos, audio, and thumbnails with advanced features.

## ✨ Features

- 📥 Download YouTube videos in multiple qualities
- 🎵 Convert videos to MP3 audio
- 🖼️ Download video thumbnails
- 🔍 Search videos by name
- ⚙️ User settings system
- 🎬 Fixed or manual quality selection
- 🧹 Automatic file cleanup
- 📊 Real-time download progress
- 🚀 Fully asynchronous

## 🛠️ Requirements

- Python 3.10+
- FFmpeg installed on system
- Telegram Bot Token
